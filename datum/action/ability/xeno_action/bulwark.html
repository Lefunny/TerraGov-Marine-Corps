Vars

affected_turfsList of turfs that are affected by this ability.
armor_keysAssociative list: [xeno] == armor_difference
armor_multiplierThe multiplier of the soft armor from the affected's base caste. This amount will be added on top of the affected's current soft armor.
channel_requiredIs the owner required to channel to use this ability? They will need to stay in the radius for the ability to continue.
flat_overhealThe flat amount of overheal to give to the affected. If it cannot be given, then it won't be subtracted from later.
overheal_keysAssociative list: [xeno] == overheal_difference
particle_holderUsed for particles. Holds the particles instead of the mob. See particle_holder for documentation.
timer_idIf any, the timer ID to check if the ability should end / maintain the ability.

Procs

apply_buffAdds any buffs to the affected xenomorph.
continue_or_endChecks if the owner can still use the ability. If yes, pay the plasma and continue. If not, end the ability.
create_affected_areaCreates the area in which the effects caused by the ability can be triggered from.
remove_affected_areaReverts any changes caused by the ability.
remove_buffRemoves any buffs from the affected xenomorph.
